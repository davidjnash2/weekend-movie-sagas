# Project Name

WEEKEND MOVIE SAGAS
## Description

Full-stack, full-CRUD MVC SPA movie app. 

Creating SQL database of multiple tables to store movie data. Retrieving data via server requests, serving to client side to be held in state and rendered to DOM.

Technologies used:
    Node
    Express
    Axios
    React
    React-Redux
    Redux-Saga
    Redux-Logger
    Javascript
    PostgreSQL
    Postico
    Postman
    VSCode
